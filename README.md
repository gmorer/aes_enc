# AES_EN
simple enc/dec - ryption using the AES intruction from intel in assembly.
The program will generate a random aes-128 key.
```bash
$ make
$ ./test data_to_encrypt

encrypted : 226b5b1b0805a9fea356918225bc8afc

```
